The shift

Do not ask the model to be honest. Limit what it can say.

Bast sits between institutional knowledge and AI output. Before anything reaches a user, the system checks scope, retrieves only eligible knowledge, scores relevance, routes to the right answering behavior, and refuses when the support is missing.

Proof before output

The answer is only the visible part.

Supported question

Answers come from approved knowledge.

The model sees only the narrow context that passed the scope and relevance checks. The response is written for the person asking, but anchored in the institution's own source material.

Unsupported question

A clean no is better than a confident guess.

If the request is out of scope or the knowledge base cannot support an answer, Bast refuses before the model improvises.

Institutional trace

The route is inspectable.

Teams can see the classification, source set, relevance score, injected context, and answer path behind a turn. The user gets clarity. The institution gets accountability.
